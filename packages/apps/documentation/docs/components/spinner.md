# Spinner

## Usage

### Small
```html:preview
<oui-spinner size="s"></oui-spinner>
```

### Medium
```html:preview
<oui-spinner size="m"></oui-spinner>
```

### Big
```html:preview
<oui-spinner size="l"></oui-spinner>
```

## API

| Attribute | Type      | Binding   | One-time binding  | Values        | Default   | Description
| ----      | ----      | ----      | ----              | ----          | ----      | ----
| `size`    | string    | @?        | yes               | `s`, `m`, `l` | `m`       | Spinner size
