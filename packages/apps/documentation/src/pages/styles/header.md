# Header

## Usage

### Simple

```html:preview
<header class="oui-header">
    <div class="oui-header__container">
        <div class="oui-header__content">
            <h1 class="oui-header__title">Title</h1>
        </div>
    </div>
</header>
```

### Simple with description

```html:preview
<header class="oui-header">
    <div class="oui-header__container">
        <div class="oui-header__content">
            <h1 class="oui-header__title">Title</h1>
            <h2 class="oui-header__description">Description</h2>
        </div>
    </div>
</header>
```

### Simple with tabs

```html:preview
<header class="oui-header">
  <div class="oui-header__container">
    <div class="oui-header__content">
      <h1 class="oui-header__title">Title</h1>
      <h2 class="oui-header__description">Subtitle</h2>
    </div>
  </div>
  <div class="oui-header__tabs">
    <nav class="oui-header-tabs">
      <ul class="oui-header-tabs__container">
        <li class="oui-header-tabs__item">
          <a href="#!/ovh-ui-kit">Home</a>
        </li>
        <li class="oui-header-tabs__item oui-header-tabs__item_active">
          <a href="#!/ovh-ui-kit/header-tabs">Header Tabs</a>
        </li>
        <li class="oui-header-tabs__item">
          <a href="#!/ovh-ui-kit/header">Header</a>
        </li>
      </ul>
    </nav>
  </div>
</header>
```
